Member Enumeration Documentation

◆ Type

enum uds::DiagReturnCode::Type

Return codes used throughout diagnosis framework.

Note
* The prefix ISO_ indicates that this is an official ISO14229 code.

@important The DiagReturnCode::Type NOT_RESPONSIBLE has a special meaning. Returning any other value from verify indicates that the job has taken control over the request and will handle it. No further job will be notified of the request in question!

Enumerator
NOT_RESPONSIBLE 

The job verifying a request is not responsible for handling it

OK 

The job wants to handle the request
