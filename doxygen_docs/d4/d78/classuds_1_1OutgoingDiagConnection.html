◆ isConflicting()

bool uds::OutgoingDiagConnection::isConflicting ( OutgoingDiagConnection const &  connection)

Checks is a connection conflicts with this OutgoingDiagConnection.

Parameters
connectionOutgoingDiagConnection to check for conflicts

A conflict arises if both connections have the same serviceId and both may receive negative responses because a negative response contains only the serviceId and this must be used to dispatch a incoming response to a distinct OutgoingDiagConnection (which is impossible with more than one connection having the same serviceId).
