Detailed Description

template<size_t N>
class io::JoinReader< N >

Class which wraps (joins) N readers into one single reader interface.

The reader will return from each of the readers in sequence.

[TPARAMS_BEGIN]

Template Parameters
NNumber of IReaders this JoinReader wraps. [TPARAMS_END]

template<size_t N>
etl::span< uint8_t > io::JoinReader< N >::peek
overridevirtual

Returns a slice to the next piece of available data.

The strategy used is a form of round-robin, where one element is returned from each queue in turn. Note that the same element will keep getting returned until release() is called.

Returns
- Slice of bytes if one of the underlying queues was not empty
  • Empty slice otherwise
