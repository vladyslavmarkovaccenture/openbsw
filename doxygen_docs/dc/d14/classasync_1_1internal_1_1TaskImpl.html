Detailed Description

template<class Adapter, ContextType Context, size_t StackSize = 0U>
class async::internal::TaskImpl< Adapter, Context, StackSize >

Primary template class that serves as a container for task-related objects and configuration.

TaskImpl provides a structure to hold essential components for tasks, such as the stack, task object, and timer object, along with configuration settings. It serves as a base for derived task classes (e.g., Task, IdleTask) and is designed for integration with FreeRTOS, using the Adapter type for flexibility.

Template Parameters
AdapterThe adapter type that supplies specific task configuration types and functions.
ContextThe context in which the task operates.
StackSizeThe size of the stack allocated for the task.
