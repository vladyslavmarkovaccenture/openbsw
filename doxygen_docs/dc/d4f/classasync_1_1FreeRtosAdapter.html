Detailed Description

template<class Binding>
class async::FreeRtosAdapter< Binding >

Adapter class bridging FreeRTOS functionalities with the application's binding.

The FreeRtosAdapter class serves as a centralized interface for managing tasks, timers, and scheduling functionalities within a FreeRTOS environment. It utilizes the specified Binding type to adapt and configure task-related components, such as TaskContext, TaskConfig, and idle and timer tasks.

Template Parameters
BindingThe binding type specifying application-specific configurations.
