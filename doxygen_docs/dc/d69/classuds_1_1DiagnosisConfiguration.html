Detailed Description

template<uint8_t NUM_INCOMING_CONNECTIONS, uint8_t NUM_OUTGOING_CONNECTIONS, uint8_t MAX_NUM_INCOMING_MESSAGES>
class uds::DiagnosisConfiguration< NUM_INCOMING_CONNECTIONS, NUM_OUTGOING_CONNECTIONS, MAX_NUM_INCOMING_MESSAGES >

Configuration for a diagnosis layer

Template Parameters
NUM_INCOMING_CONNECTIONSnumber of incoming connections that are handled by the layer (i.e. simultaneous requests from outside to the layer)
NUM_OUTGOING_CONNECTIONSnumber of outgoing connections provided by the layer (i.e. number of simultaneous request that applications may use to send requests
MAX_NUM_INCOMING_MESSAGESthis parameter specifies the maximum number of TransportMessages that can be queued inside the diagnosis layer. For a functional request this number should equal the number of TransportMessages available in the system.
