Detailed Description

A diagnosis connection representing an incoming request.

Identifiers

The expression Identifiers used in addIdentifier(), getIdentifier() and getNumIdentifiers() are NOT to be confused with the the identifiers of the ISO14229 specification. They are simply a facility to add bytes of a incoming request to the response that is sent through the IncomingDiagConnection.

Example

Given the request 0x31,0x01,0x01,0x02, adding the identifiers 0-3 will result in the prefix 0x71,0x01,0x01,0x02 being automatically added to a positive response.
