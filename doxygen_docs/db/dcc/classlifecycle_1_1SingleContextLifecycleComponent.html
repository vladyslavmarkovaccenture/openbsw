Detailed Description

A LifecycleComponent base class that allows to specify a single async context in which all transitions are run.

A component implementing SingleContextLifecycleComponent must provide implementations for the three state transition functions init, run and shutdown. After registering a LifecycleComponent to the LifecycleManager, the LifecycleManager will make sure that these are executed at the appropriate time.
