Detailed Description

Class representing a CANFrame.

An object of type CANFrame does not provide any payload-buffer by default. It has to be provided with buffers by using a certain constructor or the setPayload() method.

Note
* CANFrame may be used as base class for special frame classes providing the programmer with explicit methods to access the signals that are encoded in the CANFrames payload.
