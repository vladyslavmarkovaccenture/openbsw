Detailed Description

template<size_t CAPACITY, size_t MAX_ELEMENT_SIZE, typename SIZE_TYPE = uint16_t>
class io::MemoryQueue< CAPACITY, MAX_ELEMENT_SIZE, SIZE_TYPE >

Lock free single producer single consumer queue of variable size slices. [TPARAMS_BEGIN]

Template Parameters
CAPACITYNumber of bytes that this MemoryQueue shall provide.
MAX_ELEMENT_SIZEMaximum size of one allocation
SIZE_TYPEType used to store size of allocation internally [TPARAMS_END]

overhead

In order to manage the internal storage, the MemoryQueue introduces an overhead of sizeof(SIZE_TYPE) bytes per allocation.

Alignment

The current implementation makes no assumptions about alignment. This is why the size of an allocation is serialized using ::etl::unaligned_type<>.

Concurrency

This MemoryQueue is designed as a lock free single producer single consumer queue.
