Checks the given value.

Checks if the given value matches the one which was passed to the constructor (expectedValue). If it doesn't, the handler gets called.

Note
The handler method is called from the same context as this method.
Parameters
[in]valueThe value to check
[in]contextThe context to be stored within the monitor. It is copied to an internal member variable. Therefore the caller doesn't have to worry about the references lifetime.
