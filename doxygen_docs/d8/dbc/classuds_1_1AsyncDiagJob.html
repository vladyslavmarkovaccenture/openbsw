Detailed Description

template<class T>
class uds::AsyncDiagJob< T >

Template function that encapsulates an asynchronous diagnostic job, makes it capable of storing parallel requests and execute the process() function from within a specified context

Note
The AsyncDiagJob class needs to derive from the diagnostic job class to reliably receive the responseSent to trigger processing of stored requests.

To support any kind of diag job to encapsulate the asynchronous class derives from the specified base class.
