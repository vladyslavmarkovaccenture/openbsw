Detailed Description

template<class Binding>
class async::TaskContext< Binding >

Provides an interface between application-specific Tasks and Timers and the FreeRTOS framework, managing FreeRTOS* task and timer callbacks.

The TaskContext class facilitates seamless integration between the application's task and timer logic and FreeRTOS by invoking necessary FreeRTOS functions for task creation, scheduling and processing callbacks.
