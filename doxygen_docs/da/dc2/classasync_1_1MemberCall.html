Detailed Description

template<typename Handler, void(Handler::*)() handleFunc>
class async::MemberCall< Handler, handleFunc >

A template class that lets specify member function of eventually non-runnable class to be called as result of async::execute(), async::schedule() or async::scheduleAtFixedRate(), this way mimicking non-runnable class to act as runnable.

Template Parameters
HandlerThe type of (non-runnable) class.
handleFuncThe member function to be called.
