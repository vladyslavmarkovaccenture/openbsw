Detailed Description

A LifecycleComponent base class where all transitions will be run in the LifecycleManager's async context.

A component implementing SimpleLifecycleComponent must provide implementations for the three state transition functions init, run and shutdown. After registering a LifecycleComponent to the LifecycleManager, the LifecycleManager will make sure that these are executed at the appropriate time.
