Detailed Description

template<size_t EventCount, typename Lock>
class async::EventDispatcher< EventCount, Lock >

A template class that allows a handler function to be registered or deregistered, and then executed based on the provided eventMask. Each event corresponds to a bit in the eventMask. The handler function is executed when the specified bit in the eventMask is set.

Template Parameters
EventCountNumber of events to be handled.
LockRAII lock to support concurrent execution.
