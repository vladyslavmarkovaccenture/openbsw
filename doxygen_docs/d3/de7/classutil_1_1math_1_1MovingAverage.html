Detailed Description

template<class T, uint8_t N>
class util::math::MovingAverage< T, N >

Based on the tiny_ring container implementation of the moving average algorithm.

This class should be initialized by the threshold value in the constructor. All elements of the class will be equal to this value after the initialization. First added element will move the average result below or above the threshold.

Maximum number of stored elements is 256. This is a limitation of the tiny_ring container.
