User Documentation

Overview

The util module is a collection of various utilities and ready-to-use solutions that may be directly integrated in the client code as util library, including commands for terminal, CRC algorithms, logging commands, memory manipulation, user managed memory, streaming utilities, and string wrapper. The util module is intensively used throughout the project by other modules or application code and is subject to update and expansion.

Submodules

  • util::assert- assert

  • util::command - command framework

  • util::crc - cyclic redundancy check

  • util::format - printf-like string formatting

  • util::defere - deferred function execution

  • util::logger - logging functionality

  • util::memory - memory management

  • util::meta - miscellaneous

  • util::stream - data i/o

  • util::string - string handling
